About: A first course in robotics. The course emphasizes learning through extensive numerical simulations and animations. Mathematical modeling and theory is explained to build the necessary intuition to develop numeric code. Python code provides examples on how to program the physics, develop controllers, and animate the system. This material is supplemented with MuJoCo (version 2.2.1) in python (a free physics simulator). MuJoCoPy material may be accessed here: tiny.cc/mujocopy. I recommend installing atom IDE and python 3. Instructions can be found here:https://pab47.github.io/mujocopy.html#lec2. If you are looking for Robotics with MATLAB see this link: tiny.cc/robotics.
