About: A first course in 2D robotics. The material includes systems such as: manipulators, differential drive car, hopper, and bicopter and techniques such as: forward/inverse kinematics, euler-lagrange dynamics, linear and nonlinear control, trajectory generation, trajectory optimization, and motion planning. The course emphasizes learning through extensive numerical simulations and animations. Mathematical modeling and theory is explained to build the necessary intuition to develop numeric code. Python code provides examples on how to program the physics, develop controllers, and animate the system. I recommend installing Visual Studio Code IDE for python (see top of the table for instructions on how to install). If you are looking for Robotics with MATLAB see this link: tiny.cc/robotics. If you are looking for Robotics with Python and MuJoCo see this link: tiny.cc/roboticspy.
